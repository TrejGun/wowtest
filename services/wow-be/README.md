# WOW BE Service

## Installation

Copy enviroment file

```bash
cp .env.sample .env.development
```

Install npm modules
```bash
npm i
```

Start in watch mode
```bash
npm run start
```

Start in production mode
```bash
npm run build
npm run start:prod
```

## Swagger

Swagger ui is available on

```
http://localhost:3001/swagger
```
